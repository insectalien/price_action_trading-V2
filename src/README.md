台股價格行為 (Price Action) 自動化回測系統
這是一個基於 Python 開發的量化交易回測工具，專門用於分析台股（以台積電 2330 為例）的「吞沒形態 (Engulfing Pattern)」策略。
## 核心功能
- 自動化數據抓取：整合 FinMind API 並具備本地 CSV 緩存機制。
- PA 策略邏輯：結合吞沒形態、20日均線 (MA20) 趨勢過濾與成交量驗證。
- 進階回測引擎：支援動態停損 (Stop Loss)、停利 (Take Profit) 以及反向訊號離場邏輯。
- 參數優化實驗：可自動化測試不同風險參數對勝率的影響。
- 工業級架構：採用 OOP 物件導向設計、模組化結構與 Logging 日誌系統。

## 回測表現 (台積電 2330)
根據本系統之參數實驗，在台積電標的上：
- 停損點：10%
- 勝率：80.00%
- 累積報酬率：102.61%

## 專案結構
- `src/`: 核心原始碼（資料載入、策略邏輯、回測引擎、視覺化）
- `data/`: 本地緩存數據
- `main.py`: 專案啟動入口
- `requirements.txt`: 依賴套件清單
